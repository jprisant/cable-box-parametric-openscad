# Code of Conduct

## Our Standard

Use respectful, constructive communication. Keep collaboration technical and evidence-based.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Doxxing, threats, or abusive language.
- Repeated bad-faith disruption.

## Enforcement

Project maintainers may remove comments or contributions that violate this policy.

## Reporting

Report concerns privately to the repository maintainer.
