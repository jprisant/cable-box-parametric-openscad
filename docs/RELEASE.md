# Release Checklist

- [ ] Verify OpenSCAD model renders for default profile.
- [ ] Run smoke checks (`scripts/scad-smoke.ps1` or `scripts/scad-smoke.sh`), or verify GitHub workflow pass.
- [ ] Validate one physical print for fit and wall integrity.
- [ ] Confirm docs reflect parameter/behavior changes.
- [ ] Update `CHANGELOG.md`.
- [ ] Verify license and third-party notices are current.
- [ ] Tag release.
