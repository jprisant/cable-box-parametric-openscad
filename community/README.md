# Community

Community-contributed builds, remixes, and showcases live here.

## What belongs here

- User variants and remixes
- Print photos and setup shots
- Alternative parameter profiles
- Build notes and lessons learned

## Suggested contribution layout

- `community/<github-user-or-handle>/<project-name>/README.md`
- `community/<github-user-or-handle>/<project-name>/images/*`
- `community/<github-user-or-handle>/<project-name>/stl/*` (optional)
- `community/<github-user-or-handle>/<project-name>/config.*` (recommended)

## Contributor README minimum

- Short description and use case
- Parameter highlights
- Printer/material used
- Known fit adjustments
- License/attribution notes if third-party content is included

## Moderation and quality guidance

- Keep content respectful and technically useful.
- Avoid storing sensitive personal information in media or docs.
- Verify that contributed files are license-compatible with the repository policy.
