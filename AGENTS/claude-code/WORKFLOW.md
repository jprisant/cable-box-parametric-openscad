# Claude Code Workflow

## Start Session

1. Read `AGENTS/README.md`.
2. Read `AGENTS/claude-code/CONTEXT.md`, `TODO.md`, and `DECISIONS.md`.
3. Confirm active scope from `AGENTS/claude-code/INDEX.md`.

## During Work

- Keep project truth in source/docs, not only in agent notes.
- Capture decisions briefly with rationale.
- Keep planning artifacts in `PLANNING/` and session summaries in `SESSION-LOG/`.

## End Session

1. Update `TODO.md` with next actions.
2. Add a session note in `SESSION-LOG/`.
3. Refresh `INDEX.md` with clear continuation prompt using `AGENTS/HANDOFF_TEMPLATE.md`.
