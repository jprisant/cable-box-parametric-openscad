# Codex Workflow

## Start Session

1. Read `AGENTS/README.md`.
2. Read `AGENTS/codex/CONTEXT.md`, `TODO.md`, and `DECISIONS.md`.
3. Confirm current priority in `AGENTS/codex/INDEX.md`.

## During Work

- Keep changes small and verifiable.
- Update canonical project docs when behavior changes.
- Record durable choices in `DECISIONS.md`.

## End Session

1. Update `TODO.md`.
2. Add session summary in `SESSION-LOG/`.
3. Update `INDEX.md` with handoff details using `AGENTS/HANDOFF_TEMPLATE.md`.
